About Uni Scout

Discover the unified smart campus portal driving interaction and engagement

Uni Scout Smart Portal

v2.0.0 Stable

Uni Scout is a dynamic next-generation campus resource directory and information hub designed to unify disparate academic and extra-curricular campus systems. By streamlining notifications, notices, certificate issuances, clubs dashboards, and department directories, U-Scout drives student collaboration, minimizes administrative bottlenecks, and boosts overall campus performance.

System Specifications & Architecture

Frontend

Semantic HTML5, Vanilla ES6 JS, Flex & CSS Grid Core Layouts

Backend

Express Node JS, CORS Middleware, JSON API Protocols

Database

Relational SQL Database (MySQL Engine via mysql2)
